#  FASHION CULTURE

A modern fashion e-commerce website built using **Flask**, **HTML**, **CSS**. The website provides a clean and responsive shopping experience for fashion lovers.

## 🌐 Live Demo

🔗 https://fashion-culture-1.onrender.com

---

## ✨ Features

- 🛍️ Modern Homepage
- 👟 Shoes Collection
- 👕 T-Shirts Collection
- ⌚ Watches Collection
- 🌸 Perfume Collection
- 🔍 Product Categories
- 📱 Responsive Design
- 📞 Contact Us Page
- 🎨 Attractive UI
- ⚡ Fast Loading

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/priyamgames2-prog/FASHION-CULTURE.git
```

Move into the project

```bash
cd FASHION-CULTURE
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```

Open in your browser

```
http://127.0.0.1:5000
```
